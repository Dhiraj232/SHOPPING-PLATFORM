# 🛍️ Nayra - E-Commerce Shopping Platform

Nayra is a modern, responsive e-commerce web application inspired by popular online shopping platforms. It provides users with a smooth and intuitive shopping experience, including product browsing, cart management, and secure user authentication.

---

## 🚀 Features

* 🔐 User Authentication (Login / Signup using Firebase)
* 🛒 Add to Cart & Remove from Cart
* 📦 20–25 Dress Products with images, price & discounts
* 🔍 Product Search Functionality
* 🎯 Product Filtering (Price Low → High / High → Low)
* 💳 Checkout System (Dummy / Payment Ready)
* 📱 Fully Responsive Design (Mobile + Desktop)
* ⭐ Product Ratings & Discount Badges

---

## 🧱 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Firebase Authentication & Firestore
* **Database:** Firebase Firestore

## 🔧 Setup Instructions

1. Clone the repository:

```
git clone https://github.com/your-username/nayra.git
```

2. Open the project folder:

```
cd nayra
```

3. Setup Firebase:

* Go to Firebase Console
* Create a new project
* Enable Authentication (Email/Password)
* Enable Firestore Database
* Copy your Firebase config

4. Paste Firebase config in:

```
js/firebase.js
```

5. Run the project:

* Open `index.html` in your browser
